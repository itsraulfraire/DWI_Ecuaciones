# Calculadora Digital para Ecuaciones Diferenciales, Derivadas e Integrales

## Objetivo del proyecto

Desarrollar una aplicación web que permita a los usuarios realizar cálculos matemáticos relacionados con ecuaciones diferenciales, derivadas e integrales de manera rápida y sencilla. El objetivo es proporcionar una herramienta de apoyo académico para estudiantes y docentes, facilitando la resolución y comprensión de problemas matemáticos mediante una interfaz accesible y fácil de usar. Además, el proyecto tiene como finalidad aplicar buenas prácticas de control de versiones utilizando Git y GitHub mediante el uso de ramas, commits, pull requests y resolución de conflictos.

## Flujo de trabajo utilizado

1. Se creó un repositorio público en GitHub para alojar el proyecto.
2. Se subió la versión inicial de la aplicación web a la rama principal (`main`).
3. Se creó la rama `dev` para desarrollar nuevas funcionalidades relacionadas con los cálculos matemáticos.
4. Se creó la rama `diseno` para realizar mejoras visuales y ajustes en la interfaz de usuario.
5. Cada modificación se registró mediante commits con mensajes claros y descriptivos.
6. Los cambios desarrollados en la rama `dev` fueron enviados a GitHub y se creó un Pull Request hacia la rama principal.
7. Se realizó una revisión de código simulada por parte del integrante asignado como integrador.
8. Después de la aprobación, los cambios fueron integrados a la rama principal mediante un merge.
9. Se generó un conflicto entre ramas al modificar un mismo archivo y posteriormente se resolvió manualmente.
10. Finalmente, se verificó el correcto funcionamiento de la aplicación y se actualizó la documentación del proyecto.

## Estructura de ramas

- `main`: Rama principal del proyecto.
- `dev`: Rama utilizada para el desarrollo de nuevas funcionalidades.
- `diseno`: Rama utilizada para modificaciones de diseño y experiencia de usuario.

## Características de la aplicación

- Cálculo de derivadas.
- Cálculo de integrales.
- Resolución de ecuaciones diferenciales.
- Interfaz web intuitiva y fácil de utilizar.
- Visualización clara de resultados matemáticos.
- Diseño adaptable para diferentes dispositivos.
