# Escuela de Artes Marciales Seiryukan

## Objetivo del proyecto

Desarrollar una página web para una escuela de artes marciales Seiryukan que permita a los usuarios conocer la información de la institución, consultar los cursos disponibles, leer opiniones de alumnos, obtener información sobre las clases y adquirir material necesario para el entrenamiento. Además, el proyecto tiene como finalidad aplicar buenas prácticas de control de versiones utilizando Git y GitHub mediante el uso de ramas, commits, pull requests y resolución de conflictos.

## Flujo de trabajo utilizado

1. Se creó un repositorio público en GitHub para alojar el proyecto.
2. Se subió la versión inicial de la página web a la rama principal (`main`).
3. Se creó la rama `dev` para desarrollar nuevas funcionalidades y contenido del sitio.
4. Se creó la rama `diseno` para realizar cambios relacionados con la apariencia visual y estilos CSS.
5. Cada modificación se registró mediante commits con mensajes claros y descriptivos.
6. Los cambios realizados en la rama `dev` fueron enviados a GitHub y se creó un Pull Request hacia la rama principal.
7. Se realizó una revisión de código simulada por parte del integrante asignado como integrador.
8. Después de la aprobación, los cambios fueron integrados a la rama principal mediante un merge.
9. Se generó un conflicto entre las ramas de desarrollo al modificar el mismo archivo y posteriormente se resolvió manualmente.
10. Finalmente, se verificó el correcto funcionamiento del proyecto y se actualizó la documentación del repositorio.

## Tecnologías utilizadas

- HTML5
- CSS3
- Git
- GitHub

## Estructura de ramas

- `main`: Rama principal del proyecto.
- `dev`: Rama utilizada para el desarrollo de nuevas funcionalidades.
- `diseno`: Rama utilizada para modificaciones de diseño y estilos.

## Características del sitio web

- Información general de la escuela de artes marciales.
- Visualización de cursos y clases disponibles.
- Sección de opiniones y testimonios de alumnos.
- Información sobre instructores y actividades.
- Catálogo de material para entrenamiento y práctica.
- Diseño web responsivo y amigable para el usuario.
